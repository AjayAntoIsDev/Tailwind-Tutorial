Tailwind CSS Tutorial for Class 11th

Welcome to Tailwind CSS! üé®

Tailwind CSS is a utility-first CSS framework that lets you build beautiful websites quickly without writing custom CSS. Instead of writing CSS files, you add classes directly to your HTML elements!

What you'll learn: HTML basics recap, Tailwind classes, layouts, components, and build a complete website!

HTML Tags - Quick Recap üìö

<div> - Division/Container

Used to group and organize content. It's a block-level element.

<div>This is a container</div>

<h1> to <h6> - Headings

Heading tags from largest (h1) to smallest (h6).

<h1>Main Heading</h1> <h2>Sub Heading</h2>

<p> - Paragraph

Used for text content and paragraphs.

<p>This is a paragraph of text.</p>

<a> - Anchor/Link

Creates clickable links to other pages or sections.

<a href="https://example.com">Click here</a>

<img> - Image

Displays images. It's a self-closing tag.

<img src="image.jpg" alt="Description">

<button> - Button

Creates clickable buttons.

<button>Click Me</button>

<nav> - Navigation

Contains navigation links for the website.

<nav>Navigation menu goes here</nav>

<section> - Section

Defines a section in a document.

<section>Content section</section>

Getting Started with Tailwind CSS üöÄ

To use Tailwind CSS, add this line in your HTML <head> section:

<script src="https://cdn.tailwindcss.com"></script>

That's it! Now you can use all Tailwind classes in your HTML.

2Ô∏è‚É£ Spacing (Padding & Margin)

Padding

Padding adds space inside an element:

<div class="p-4">Padding on all sides</div> <div class="px-6">Padding left and right only</div> <div class="py-8">Padding top and bottom only</div> <div class="pt-4 pb-2">Different padding top and bottom</div>

Result:

Padding on all sides (p-4)

Padding left and right (px-6)

Padding top and bottom (py-8)

Margin

Margin adds space outside an element:

<div class="m-4">Margin on all sides</div> <div class="mx-auto">Center element horizontally</div> <div class="mt-8">Margin top only</div>

Common spacing values:

  • p-2 = 0.5rem (8px)
  • p-4 = 1rem (16px)
  • p-6 = 1.5rem (24px)
  • p-8 = 2rem (32px)

üìñ Quick Reference Guide

Colors

  • ‚Ä¢ text-{color}-{shade} - Text color
  • ‚Ä¢ bg-{color}-{shade} - Background color
  • ‚Ä¢ Colors: blue, red, green, purple, yellow, gray
  • ‚Ä¢ Shades: 100-900 (lighter to darker)

Spacing

  • ‚Ä¢ p-{size} - Padding all sides
  • ‚Ä¢ px-{size} - Padding left/right
  • ‚Ä¢ py-{size} - Padding top/bottom
  • ‚Ä¢ m-{size} - Margin (same pattern)

Text

  • ‚Ä¢ text-{size} - xs, sm, base, lg, xl, 2xl, 3xl
  • ‚Ä¢ font-{weight} - light, normal, semibold, bold
  • ‚Ä¢ text-{align} - left, center, right

Layout

  • ‚Ä¢ flex - Flexbox container
  • ‚Ä¢ grid grid-cols-{n} - Grid layout
  • ‚Ä¢ justify-{type} - center, between, around
  • ‚Ä¢ items-center - Vertical center

Borders & Corners

  • ‚Ä¢ border - Add border
  • ‚Ä¢ border-{size} - 2, 4, 8
  • ‚Ä¢ rounded - Small corners
  • ‚Ä¢ rounded-{size} - lg, xl, full

Effects

  • ‚Ä¢ shadow-{size} - sm, md, lg, xl
  • ‚Ä¢ hover:bg-{color} - Hover effect
  • ‚Ä¢ opacity-{value} - 0-100
